# Math-to-LaTeX Converter 

A Python projects that converts raw English language into LaTeX format.

## Features

- Functional input that gets user's text e.g., pi, sum from 1 to 10, 5 over 10, theta, beta, gamma, x+5 / y+4, 2 raise to 3.
- A Field that displays LaTeX structure to be copied and pasted.
- Mathematical LaTeX preview. 
- Displays Mathematical LaTeX function e.g., pi.

## Installation

```bash
pip install streamlit
git clone <repository-url>
cd Math-to-LaTeX Converter
```

## Clone this Project

git clone [https://github.com/Nemejohn/math-to-latex-converter](https://github.com/Nemejohn/math-to-latex-converter)

```bash
cd math-to-latex
```

## Usage

```bash
streamlit run your_filename.py
```

Open your browser and navigate to `http://localhost:8501`
